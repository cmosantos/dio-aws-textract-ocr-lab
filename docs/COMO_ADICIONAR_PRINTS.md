# Como registrar as evidências do laboratório

Depois de executar o projeto com suas credenciais AWS, tire dois prints:

1. O terminal mostrando o texto identificado e os caminhos dos arquivos gerados.
2. A tela do Amazon Textract ou o conteúdo do arquivo de texto extraído.

Salve as imagens na pasta `assets` com estes nomes:

- `execucao_terminal.png`
- `resultado_textract.png`

Em seguida, adicione ao README:

```markdown
![Execução do projeto](../assets/execucao_terminal.png)

![Resultado do Textract](../assets/resultado_textract.png)
```

Nunca publique Access Key, Secret Access Key, tokens, e-mails corporativos, IDs sensíveis ou dados pessoais.
